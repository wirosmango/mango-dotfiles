# Dependencies
```
mangowm (AUR), rofi, mako, fastfetch, waybar, swaybg, hyprshot, kitty, thunar, ttf-jetbrains-mono, otf-font-awesome, cliphist, wiremix, wlogout (AUR), bluetui (AUR), swaylock-effects (AUR)
```

# **Installation**
```
git clone https://github.com/wirosmango/mango-dotfiles.git
cd mango-dotfiles
chmod +x install.sh
./install.sh
```
